# ✈️ Flight Booking Backend

A backend API built using **Node.js** and **Express.js** for managing a flight booking system.

## 🚀 Features
- 🔐 Secure user authentication using bcrypt
- 🗄️ Flight booking management with MongoDB
- 📧 Automatic booking confirmation emails using Nodemailer
- ⚙️ Environment variables handled with .env for security

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB
- bcrypt
- Nodemailer

## 📂 Project Setup

1. Clone the repository
git clone https://github.com/Ombhudhara/Flightbookingwebsitemaster.git

2. Install dependencies
npm install

3. Create a `.env` file and add required credentials

4. Start the server
npm start
